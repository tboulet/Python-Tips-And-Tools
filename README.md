# Python-Tips-And-Tools
Repo for learning and referencing usefull python tips and tricks or use of libraries.

## How to read this repo ?
This repo contains notebooks from which you can learn python tips and tricks, and folders that contain a README to read as well as files necessary for the code to run. 

If a folder or a sub-folder name starts with 'project_' it means it has to be considered by the reader as a project folder (for working on one particular task) or a GitHub repository.
